# 01-2 傳1個變數

### 1. 範例下載
```
<下載>
  |__ web.zip
```

### 2. 檔案放置方式
```
 <web>
   |
   |__ <static>
   |      |__ <css>
   |      |     |__ main.css   (使用範例檔案)
   |      |     |__ menu.css   (使用範例檔案)
   |      |     |__ color.css  (使用範例檔案)
   |      |
   |      |__ <imgs>
   |            |__ header.jpg  (使用範例檔案)
   |
   |__ <templates>
   |      |__ index.html   (使用範例檔案)
   |      
   |__ app.py  
```


### 3. 測試版本
```
Python 3.8.1 32-bit
```


### 4. 追加外掛
```
pip install flask
```

### 5. app.py
``` py
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html', version='測試版 1.0')

if __name__ == '__main__':
    app.run(debug=True, port=3000, threaded=True)
```


### 6. index.html
``` html
    .
    .
    .

        <div class="content">
            <h2>樣板畫面 - {{version}}</h2>
        </div> 
        
    .
    .
    .        
```

### 7. 測試
```
http://localhost:3000

(如要修改了程式要重載新內容, 請按著[shift]鍵, 同時按瀏覽器的重整畫面)
```
