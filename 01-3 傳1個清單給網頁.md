# 01-3 傳1個清單給網頁

### 1. 範例下載
```
<下載>
  |__ web.zip
```

### 2. 檔案放置方式
```
 <web>
   |
   |__ <static>
   |      |__ <css>
   |      |     |__ main.css   (使用範例檔案)
   |      |     |__ menu.css   (使用範例檔案)
   |      |     |__ color.css  (使用範例檔案)
   |      |
   |      |__ <imgs>
   |            |__ header.jpg  (使用範例檔案)
   |
   |__ <templates>
   |      |__ index.html           (使用範例檔案)
   |      |__ customer_list.html   (由「子功能畫面樣版」修改)   
   |      
   |__ app.py  
```


### 3. 測試版本
```
Python 3.8.1 32-bit
```


### 4. 追加外掛
```
pip install flask
```
